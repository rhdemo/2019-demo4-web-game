## Usage

Clone the repository and install the dependencies:

```sh
git clone --depth 1 git@github.com:rhdemo/2019-demo4-web-game.git
yarn
```

### Development

```
yarn start:dev
```

### Production

```
yarn build
```

### Testing

```
yarn test
```
